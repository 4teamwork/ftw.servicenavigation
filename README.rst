Installation
============

Add the package as dependency to your setup.py:

.. code:: python

  setup(...
        install_requires=[
          ...
          'ftw.servicenavigation',
        ])

or to your buildout configuration:

.. code:: ini

  [instance]
  eggs += ftw.servicenavigation

and rerun buildout.
